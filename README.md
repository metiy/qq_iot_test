# 使用 QQ 控制的音乐播放器

这是一个自己玩的 QQ 物联网平台的简单例子

# 需要准备的工作

* 基于树莓派3开发，系统是 respbian ，
* QQ IOT 版本是 飞思卡尔 Freescale i.MX6 gcc4.9.1 ARM 1.6.205	
* 使用 QQ 消息进行控制
* 消息通信使用了 Redis
* 音乐播放使用了 MPD
* 编译之前，请安装好相关的软件和开发工具
* 使用之前 ，请做好如下工作
   * 申请 QQ IOT 的帐号
   * 下载产品公钥（）
   * 生成自己签名用的公私钥
   * 上面说的是2个事情，请不要搞混
   * 配置 QQ IOT 平台 ，不配置的话 ， 无法接收设备消息
   * 已经绑定后 ， 配置 QQ ， 否则也无法接收设备消息
